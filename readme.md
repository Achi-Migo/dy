## 简介

1. 抖音的点赞数，作品数是某字体显示为数字。目的是防止抓取

2. 这个简单demo基于node.js实现抓取抖音的防抓数据


## 思路

1. 截图下数据发送到AI接口识别数字返回数据


## 依赖

1. puppeteer

2. 百度AI接口

3. pm2

## 使用

cd dy

pm2 start dy -i 4   // 可以借助pm2实现多进程抓取
